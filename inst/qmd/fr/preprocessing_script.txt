## SUPPRESSION DES VARIABLES CLASSEES COMME ID OU D

> Le code ci-dessous identifie les variables à supprimer selon leur classification et les retire du jeu de données.

**Explication étape par étape**

-   **Identifier les variables à supprimer**\
    Dans le fichier de classification des variables, les variables dont la classification est **"DI"** (Identifiant direct) ou **"D"** (à suprimmer pour d'autres raison) sont sélectionnées.

    **Extraire leurs noms**\
    Les noms de ces variables sont récupérés à l’aide de `pull(Name)` afin de créer une liste de variables à supprimer.

    **Supprimer les valeurs manquantes dans la liste**\
    Les éventuelles valeurs `NA` sont retirées de cette liste.

    **Supprimer les variables du jeu de données**\
    La fonction `select(-any_of(...))` est utilisée pour **retirer ces variables du dataset** `data`.

```{r}
#| include: FALSE

variables_to_delete=variable_classification %>% filter(Classification %in% c("DI","D")) %>% pull(Name)
variables_to_delete=variables_to_delete[!is.na(variables_to_delete)]
variables_to_delete
data=data %>% dplyr::select(-any_of(variables_to_delete))

```
